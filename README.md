DAPID
===============================

DaPID support code

# Installation

First set up a conda enviroment

```
conda create -n dapid-project python=2 anaconda
```

Activate it

```
source activate dapid-project
```

Then `cd` into the project and run setup.py

```
python setup.py develop
```

# Running

Utilize `--help` to see functions

```
dapid --help
```
